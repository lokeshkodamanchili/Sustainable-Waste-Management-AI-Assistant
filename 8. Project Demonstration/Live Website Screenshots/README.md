# Stage 8: Live Website Screenshots

This directory is designated for the user interface (UI) and user journey screenshots of the live **EcoSense - Smart Sustainable Waste Assistant** web application.

---

## 📸 Recommended Screenshots to Upload Here

Please upload screenshots showing the following application states for your final evaluation:

1. **`01_Login_Page.png`**: The responsive dark-themed login interface with user input fields.
2. **`02_Dashboard_Home.png`**: The centralized system dashboard showing total items scanned, recycle success rate percentages, and telemetry cards.
3. **`03_Waste_Scanner_Plastic.png`**: The search scanner demonstrating a successful Llama 3.3 classification card for a recyclable item (e.g. plastic bottle) showing steps and eco recommendations.
4. **`04_Waste_Scanner_Hazardous.png`**: The search scanner displaying warning alerts and protocols for a hazardous item (e.g. old battery).
5. **`05_Leaflet_Map_Pins.png`**: The interactive Leaflet maps display with color-coded geo-coordinate markers pointing to local recycling hubs.
6. **`06_Scan_History_Logs.png`**: The personal scan history stack showing logged items, timestamps, categories, and deletion buttons.
7. **`07_ChartJS_Analytics.png`**: The visual graphs page rendering Doughnut distributions, Bar charts, and Line graphs of scan habits.
8. **`08_Settings_Reset.png`**: The settings configuration panel with options to seed mock data datasets and reset systems.
